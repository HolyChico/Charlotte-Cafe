# Charlotte-Cafe

Acesse o resultado: https://holychico.github.io/Charlotte-Cafe/

Um projeto de site responsivo para uma cafeteria (hipotética) usando Bootstrap.

## Sobre o Projeto

O Charlotte Café é uma landing page completa que simula a presença digital de um café premium com temática francesa. O site permite que os usuários naveguem por um cardápio filtrável, conheçam a história do estabelecimento, leiam avaliações de clientes e utilizem um formulário para reserva de mesas.

### Principais Funcionalidades
* **Menu Dinâmico**: Filtros interativos para categorias de Salgados, Doces e Bebidas utilizando a biblioteca Isotope.
* **Sistema de Reservas**: Formulário estruturado com integração visual para localização via Google Maps.
* **Carrosséis Interativos**: Sliders de destaque e depoimentos de clientes utilizando Owl Carousel.
* **Design Responsivo**: Adaptado para dispositivos móveis, tablets e desktops através do Bootstrap 4.

## Tecnologias Utilizadas

* **HTML5 / CSS3**
* **Bootstrap 4**
* **JavaScript / jQuery**
* **Google Fonts** (Great Vibes e Open Sans)
* **Font Awesome**

## Personalizações e Créditos

Este projeto foi construído utilizando como base o template gratuito [Feane](https://themewagon.com/themes/free-bootstrap-4-html5-restaurant-website-template-feane/) da ThemeWagon.

As modificações manuais incluem:
* **Identidade Visual**: Adaptação da paleta de cores para tons terrosos e pastéis (#c97a36, #5e3c2d, #fdf8f3).
* **Ajustes de Layout**: Correções via Flexbox para uniformizar a altura dos cards no menu.
* **Componentes**: Estilização personalizada de botões de navegação e elementos de hover.

# Charlotte-Cafe (English Version)

Live Demo: https://holychico.github.io/Charlotte-Cafe/

A responsive website project for a (hypothetical) coffee shop built with Bootstrap.

## About

Charlotte Café is a landing page simulating the digital presence of a premium French-themed café. The site provides a complete interface for users to browse a filterable menu, learn about the establishment's history, and use a table reservation form.

### Key Features
* **Dynamic Menu**: Interactive filters for Savory, Sweet, and Drink categories using the Isotope library.
* **Booking System**: Structured form with visual location integration.
* **Interactive Carousels**: Showcasing highlights and customer testimonials using Owl Carousel.
* **Responsive Design**: Adapted for mobile, tablets, and desktops via Bootstrap 4.

## Technologies Used

* **HTML5 / CSS3**
* **Bootstrap 4**
* **JavaScript / jQuery**
* **Google Fonts** (Great Vibes and Open Sans)
* **Font Awesome**

## Customizations and Credits

This project was built using the [Feane](https://themewagon.com/themes/free-bootstrap-4-html5-restaurant-website-template-feane/) free template by ThemeWagon as a base.

Manual modifications include:
* **Visual Identity**: Adapting the color palette to earthy and pastel tones (#c97a36, #5e3c2d, #fdf8f3).
* **Layout Adjustments**: Flexbox fixes to standardize menu card heights.
* **Components**: Custom styling for navigation buttons and hover effects.
